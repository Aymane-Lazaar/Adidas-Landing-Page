# Adidas Landing Page

This is a simple HTML and CSS project for a landing page featuring Adidas products. The page includes a navigation menu, a hero section with text and an image, and responsive design for different screen sizes.

## Project Structure


- `index.html` → Main HTML file containing the structure of the landing page.
- `style web site.css` → CSS file for styling, layout, and responsive design.
- `shoes-img/shoes.png` → Product image displayed in the hero section.

## Features

- **Navigation bar** with links: Home, Men, Women, Kids.
- **Hero section** with text and product image.
- **Gradient and background effects** for modern look.
- **Responsive design** for different screen widths.
- Hover effects on navigation links.
- Smooth layout adjustments for smaller screens using media queries.
